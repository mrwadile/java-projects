## List of Projects with Descriptions and Features

### 1. To-Do List Application
**Description**: A user-friendly application to manage daily tasks efficiently.
- **Features**:
  - Add, remove, and mark tasks as complete.
  - View all tasks with their current status.
  - Save tasks to a file for persistence.

### 2. Library Management System
**Description**: A comprehensive system for managing library books and transactions.
- **Features**:

  - Generate reports on book status.

### 5. Student Grading System
